## Features 
- **Responsive Design**: Fully responsive layout that works on all devices
- **Real-time Validation**: Live form validation with instant feedback
- **Password Strength**: Visual password strength indicator
- **Confirm Password**: Real-time password confirmation validation
- **Modern UI/UX**: Clean, modern design
- **Cross-browser Compatible**: Works on all modern browsers
- **Form Sanitization**: Client-side input sanitazation and validation

## Usage ğŸ¯
1. **Access the Form**: Open the live demo or run locally
2. **Fill Out Fields**: Enter your personal information in the required fields
3. **Real-time Validation**: Watch as fields validate in real-time as you type
4. **Password Strength**: See password strength indicators update dynamically
5. **Submit**: Click submit to see form validation in action

## Form Fields 
- **First Name**: Required text input with character validation
- **Last Name**: Required text input with character validation
- **Email**: Required email input with format validation
- **Phone Number**: Optional phone input with form validation
- **Password**: Required password with strength requirements
- **Confirm Password**: Required field that must match password
